RocketMovies

Rocket movies é o front de uma aplicação que tem como objetivo servir como local para armazenar suas opiniões e avaliações de filmes que você já viu.

Totalmente estruturado com react, separando cada componente por função clara e especifica, além de ser estilizado usando StyledComponents.

React Node StyledComponents Front-End

RocketNotes

Rocket Notes é uma aplicação que tem como objetivo servir como bloco de notas aprimorado para o usuário, onde pode-se filtrar notas por tags e titulo, inserir links uteis e acessa-los assim que clica na nota.

Totalmente estruturado com react, separando cada componente por função clara e especifica, além de ser estilizado usando StyledComponents, backend feito em NodeJs, com ajuda do Express, integrando tudo com mySQL e automatizando toda a criação de tabelas com o Knex

React Node StyledComponents Full-Stack

GitFav

GitFav é uma aplicação que tem como objetivo servir como local para salvar os seus usuários favoritos do github, apenas inserindo o nick deles, recebendo dados deles diretamente do github.

Feito com HTML, CSS e Js usado principalmente para consumir a API do github e tratar seus dados, para mostra-lo de maneira facilitada e organizada ao usuário.

HTML CSS Js API GitHub Front-End

DoctorCare

Doctor care é uma landingPage para uma clinica médica, com o intuito de expor suas especialidades, converter novos clientes em potencial, além de alcançar mais visibilidade.

Totalmente estruturado com HTML, CSS e Javascript que foi usado para manipular eventos na página e deixar mais interativa, juntamente com a biblioteca scrollReveal.

HTML CSS Javascript ScrollReveal Front-End
